Performance Metrics Explanation:

- Sharpe Ratio:
    Risk-adjusted return: mean(strategy_returns) / std(strategy_returns)
    Annualized with sqrt(12) for monthly returns.
    Higher = better risk-adjusted performance.

- Max Drawdown:
    Largest peak-to-trough decline in portfolio value.
    Lower (more negative) = worse maximum loss.

- Bull/Bear Sharpe:
    Sharpe ratio calculated separately during bullish or bearish regimes.

- Volatility Factor:
    Same computation applied to a factor based on stock volatility.

- Average Cross-Sectional Dispersion:
    Measures how much individual stock returns differ within the portfolio.

- Turnover:
    Measures how much portfolio weights change month-to-month.
    High turnover → higher transaction costs.

- Concentration:
    Measures risk from overweight positions; (weights**2).sum()


GPT review:
Back-tested momentum and volatility factors → your code calculates factor values, ranks deciles, computes long/short returns.

Across a US equity universe → S&P 500 (filtered by market cap) or custom tickers.

Decile portfolios → top/bottom 10% each month.

Return dispersion → calculated monthly using dispersion = next_month_returns.std().

Drawdowns → max_drawdown() computes worst cumulative loss.

Regime sensitivity → market_regime_filter() separates bull vs bear months.

Risk-adjusted performance → Sharpe ratio computation.

Structural weaknesses → turnover and concentration functions measure trading activity and portfolio concentration.

Implementation constraints → high turnover warns of practical issues.



| Sharpe    | Meaning     |
| --------- | ----------- |
| < 0.5     | Weak        |
| 0.5 – 1.0 | Decent      |
| 1.0 – 1.5 | Good        |
| 1.5 – 2.0 | Very good   |
| > 2.0     | Exceptional |


| Max DD | Interpretation     |
| ------ | ------------------ |
| -10%   | Mild               |
| -20%   | Normal             |
| -40%   | Aggressive         |
| -60%+  | Extremely volatile |


| Dispersion  | Meaning                  |
| ----------- | ------------------------ |
| < 0.05      | Very low differentiation |
| 0.05 – 0.15 | Normal                   |
| > 0.20      | High opportunity         |



| Turnover  | Meaning   |
| --------- | --------- |
| < 0.2     | Low       |
| 0.2 – 0.5 | Moderate  |
| 0.5 – 0.8 | High      |
| > 1.0     | Very high |
